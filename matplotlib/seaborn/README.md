# Seaborn

> 一个基于matplotlib的Python数据可视化库，提供了更高层的接口用以绘制具有吸引力和信息展现力的数据统计图。

## 安装

```bash
pip install -U seaborn
```

## 依赖

- python 2.7 or 3.5+

### 强制依赖

- numpy (>=1.9.3)
- scipy (>=0.14.0)
- matplotlib (>=1.4.3)
- pandas (>=0.15.2)

## 导入

```python
import seaborn as sns
```
