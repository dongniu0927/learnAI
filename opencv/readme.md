# OpenCV

本项目用于存储在学习openCV过程中的例子代码.

## 内容列表

- [Image Process](./image_process.ipynb)

  - 读取图片: 使用`cv2.imread()`来读取图片;
  - 显示图片: 使用`cv2.imshow()`函数在一个窗口中显示图片，这个窗口自适应图片的大小;
  - 存储图片: 使用`cv2.imwrite()`函数来保存图片;
  - 完整例子: 一个使用cv2读取,显示,保存图片的完整例子;
